# WebGL User Guide

## Install chromium-browser

<NewCodeBlock tip="Linux$" type="device">

```bash

sudo apt install chromium-browser

```

</NewCodeBlock>

## Run chromium-browser

Start chromium-browser.
Input https://webglreport.com/ or https://get.webgl.org/ in the address bar, and you can see the WebGL support information.
