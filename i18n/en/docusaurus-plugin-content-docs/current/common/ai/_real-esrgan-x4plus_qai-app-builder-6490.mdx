This document explains how to run [Real-ESRGAN-x4plus](https://aihub.qualcomm.com/models/real_esrgan_x4plus?domain=Computer+Vision&useCase=Super+Resolution&chipsets=qualcomm-qcs6490-proxy) on Radxa Dragon Q6A using the QAI AppBuilder Python library.
Image Super-Resolution Model.

## Install QAI AppBuilder

:::tip
Please refer to [**QAI AppBuilder Installation Guide**](qai-appbuilder)
:::

## Run the Example

### Install Dependencies

<NewCodeBlock tip="Device" type="device">

```bash
pip3 install requests tqdm qai-hub py3_wget opencv-python torch
```

</NewCodeBlock>

### Run the Script

- Navigate to the example directory

   <NewCodeBlock tip="Device" type="device">

  ```bash
  cd ai-engine-direct-helper/samples/linux/python
  ```

    </NewCodeBlock>

- Prepare the input image. The following image is used as an example:
