# 资源汇总下载

## 系统镜像

:::tip 系统镜像说明
请根据自己的系统启动介质选择对应的系统镜像下载

- FEL 系统镜像需要进入 FEL 模式和使用 Phoenix 工具写入系统镜像
- GPT 系统镜像可以通过 Balena Etcher 工具或者 `dd` 命令写入系统镜像
  :::
